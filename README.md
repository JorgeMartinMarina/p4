# Práctica 4

Esta práctica consiste en el desarrollo de una pequeña página web de tienda online usando HTML y CSS con varias secciones conectadas entre sí.

## Estructura del proyecto

- `index.html` → página principal
- `productos.html` → catálogo de productos
- `carrito.html` → página del carrito
- `pedido.html` → página para realizar el pedido
- `marco.html` → página auxiliar de estructura
- `style.css` → estilos generales del sitio
- `imagenes/` → carpeta con las imágenes usadas en la práctica
- `videos/` → carpeta con los vídeos usados en la práctica

## Contenido

La web incluye:

- Página de inicio
- Navegación entre apartados
- Visualización de productos
- Carrito de compra
- Formulario de pedido
- Estilos CSS externos
- Recursos multimedia en carpetas separadas
